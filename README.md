# CNN-Digit-Recognizer-PyQt6
>这个小项目仅用于练习

## 网络结构
卷积层-归一化层-激活层-最大池化-全连接

## 运行环境
Python 3.10
核心库：PyTorch, PyQt6, OpenCV, NumPy

## 项目功能
- 实时手写：在窗口画板上直接书写数字。
- 智能预处理：自动裁剪多余空白，并将数字调整为标准的 $28 \times 28$ 像素居中格式。
- 快速识别：点击按钮，利用 CNN 模型瞬间输出识别结果。

## 核心逻辑
- 取图：通过内存映射获取画板像素数据。
- 处理：将图像转为黑底白字，并进行中心化对齐（防止因书写位置偏移导致识别失败）。
- 预测：将处理后的张量喂入 CNN 网络，取最大概率对应的索引作为结果。
